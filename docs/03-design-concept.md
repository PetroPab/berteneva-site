# Дизайн-концепция

## Философия

Сайт — это часть портфолио. Аудитория увидит не только 
работы, но и то, как сделан сам сайт. Поэтому он должен 
быть **выразительным, а не нейтральным**.

Каждое визуальное решение — осознанное. 
Никакой «AI-эстетики», generic-шрифтов и фиолетовых градиентов.

Главные герои страницы — **работы клиента**. 
Всё остальное служит им.

---

## Направление: Editorial / Журнальный

**Идея:** «Я не просто фрилансер — я автор».

Сайт построен по логике печатного журнала о дизайне: 
крупные обложки, нумерация разделов, тонкие линии, 
сериф-типографика, асимметрия. Каждая секция читается 
как разворот.

## Типографика

| Роль | Шрифт | Где |
|------|-------|-----|
| Display / Заголовки | **Fraunces** (variable serif) | Hero, заголовки секций |
| Body / Подзаголовки | **Inter Tight** или **Söhne** (если есть) | Параграфы, описания |
| Mono / Детали | **JetBrains Mono** | Нумерация, метки, captions |

> ⚠️ Inter Tight — допустимое исключение, это не стандартный Inter, 
> у него заметно более плотный ритм. Если есть лицензия на Söhne — 
> использовать её. В крайнем случае — **Geist** от Vercel.

**Размерная шкала (rem):**  
`0.75 / 0.875 / 1 / 1.125 / 1.5 / 2 / 3 / 4.5 / 6.5 / 9`

**Иерархия:**
- H1 (Hero): Fraunces, 6.5–9rem, weight 400, italic для акцентов
- H2 (секции): Fraunces, 3–4.5rem, weight 400
- H3 (карточки): Fraunces, 1.5–2rem, weight 500
- Body: Inter Tight, 1rem (16px), line-height 1.6
- Caption: JetBrains Mono, 0.75rem, uppercase, letter-spacing 0.08em

## Палитра

| Токен | HEX | Применение |
|-------|-----|------------|
| `--bg` | `#F4F1EA` | Основной фон (тёплый молочный) |
| `--surface` | `#FFFFFF` | Карточки, формы |
| `--ink` | `#1A1A1A` | Основной текст |
| `--ink-muted` | `#6B6B6B` | Вторичный текст, captions |
| `--rule` | `#D9D5CB` | Линии-разделители |
| `--accent` | `#C2410C` | Терракотовый акцент (CTA, hover) |
| `--accent-soft` | `#FED7AA` | Подложка под цитаты, теги |

**Правила:**
- Один акцентный цвет на весь сайт — терракотовый
- Никаких градиентов
- Контраст текста на фоне ≥ 4.5:1 (проверено: `#1A1A1A` на `#F4F1EA` = 14.8:1 ✓)

## Сетка и spacing

- **Mobile** (≤ 640px): одна колонка, padding 20px
- **Tablet** (641–1024px): 6-колоночная сетка, padding 32px
- **Desktop** (≥ 1025px): 12-колоночная сетка, max-width 1280px, padding 48px

**Spacing-шкала (px):**  
`4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96 / 128 / 192`

**Асимметрия:**  
Заголовки секций смещены влево или вправо, не центрируются. 
Карточки портфолио чередуются: одна на всю ширину, две рядом, 
снова одна — как развороты журнала.

---

## Радиусы и границы

- Радиусы: **минимальные**. `0` для большинства блоков, 
  `2px` для мелких элементов (теги, инпуты)
- Карточки портфолио — без радиусов (как фотографии в журнале)
- Границы: 1px solid `--rule`, используются как разделители разделов

## Анимация

Анимация служит смыслу. Длительность 150–300ms, easing ease-out на вход.

- **Hero:** буквы заголовка появляются с лёгким staggered fade-in (50ms задержка между словами)
- **При скролле:** блоки появляются с `opacity 0 → 1` + `translateY 24px → 0`
- **Карточки портфолио:** hover — превью медленно увеличивается (scale 1 → 1.03), название сдвигается на 4px вправо
- **Marquee:** бегущая строка с названиями проектов между секциями Portfolio и Process
- **Reduced motion:** все анимации заменяются на простой fade или отключаются

## Иконки

- Источник: **Lucide React** (один набор на весь сайт)
- Стиль: outline, толщина обводки 1.5px
- Размеры: 16 / 20 / 24 px

---

## Декоративные элементы (фирменные)

1. **Нумерация секций** — слева вверху каждой секции: 
   `01 — About`, `02 — Services`, `03 — Portfolio` (JetBrains Mono, uppercase)
2. **Тонкие линии** — 1px разделители между крупными блоками
3. **Цитаты** — крупный сериф с italic, подложка `--accent-soft`
4. **Caption под фото** — мелкий моно-шрифт, как подпись к иллюстрации в журнале

## Mood-якоря

**Что НЕ должно случиться:**
- Сайт похож на «ещё один Linear / Vercel клон»
- Сайт выглядит как сгенерированный AI-конструктором
- Технически правильный, но забываемый

**Что должно случиться:**
- Через 2 секунды понятно настроение
- Хочется задержаться и проскроллить
- Сайт сам по себе работает как кейс

**Референсы по духу (не по копированию):**
- Are.na (типографика и сдержанность)
- It's Nice That (журнальная подача)
- Pentagram website (асимметрия и крупные заголовки)
