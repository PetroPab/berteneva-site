## Единая система размеров изображений

Все изображения проектов унифицированы по двум размерам:

```
┌─────────────────────────────────────┐
│  ГЛАВНОЕ ИЗОБРАЖЕНИЕ                │
│  16/9  →  1920 × 1080 px           │
│  На странице проекта, полная ширина │
│  Также используется как обложка     │
│  в сетке на главной странице        │
└─────────────────────────────────────┘

┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│  ДОП. 1      │ │  ДОП. 2      │ │  ДОП. 3      │
│  3/2         │ │  3/2         │ │  3/2         │
│  1200 × 800  │ │  1200 × 800  │ │  1200 × 800  │
└──────────────┘ └──────────────┘ └──────────────┘
Три одинаковых → показываются в ряд на странице проекта
```

## Общие требования к файлам

- **Формат:** JPG (рекомендуется) или PNG
- **Качество JPG:** 85–90%
- **Цветовой профиль:** sRGB
- **Имена файлов:** строчные латинские буквы и дефисы (как указано ниже)
- **Куда класть:** `public/images/projects/<slug>/` и `public/images/about/`

---

## Как делать скриншоты сайтов

**Инструмент: GoFullPage** (расширение Chrome, бесплатно)

1. Открой сайт в Chrome
2. Установи ширину окна 1440px (DevTools → адаптивный режим → Desktop)
3. GoFullPage → «Capture page» → скачать PNG/JPG
4. Обрежь до нужного соотношения в любом редакторе

**Через DevTools (без расширения):**
1. `F12` → `Ctrl+Shift+P` → «Capture full size screenshot»
2. Ширина 1440px, масштаб 100%

**Мобильная версия (для доп. изображений):**
- DevTools → iPhone 14 Pro (390px)
- Сделай скриншот нужного раздела (не полную страницу)
- Скриншот будет в пропорции ~3/2 если брать часть экрана

## Проект 5 — Хопмалт (этикетки пива)

**Папка:** `public/images/projects/khopmalt/`

> Для дизайна упаковки скриншоты не нужны —
> нужна предметная съёмка. Соотношения те же.

| Файл | Размер | Соотношение | Что снять |
|------|--------|-------------|-----------|
| `khopmalt-main.jpg` | 1920 × 1080 | **16/9** | Все бутылки/банки в ряд на нейтральном фоне |
| `khopmalt-extra-1.jpg` | 1200 × 800 | **3/2** | Крупный план одной этикетки |
| `khopmalt-extra-2.jpg` | 1200 × 800 | **3/2** | Бутылки на баре или в атмосферном месте |
| `khopmalt-extra-3.jpg` | 1200 × 800 | **3/2** | Этикетки на плоской поверхности (раскладка) |

**Советы по съёмке этикеток:**
- Нейтральный фон: серый, белый, дерево, крафтовая бумага
- Мягкое боковое освещение, без прямых бликов на стекле
- Кадр «главное» — все сорта выстроены в ряд, одинаковое расстояние
- Без сильных фильтров — цвет должен читаться точно как в оригинале

## Как подключить реальные изображения

Когда файлы готовы, в `app/projects/[slug]/page.tsx` замените
`<Placeholder>` на `<Image>`. Комментарий с кодом уже есть прямо
над каждым плейсхолдером:

```tsx
// Главное изображение
<Image
  src={`/images/projects/${project.slug}/${project.main}`}
  alt={`${project.title} — главный скриншот`}
  width={1920}
  height={1080}
  className="w-full object-cover"
  priority
/>

// Дополнительные (в цикле)
<Image
  src={`/images/projects/${project.slug}/${filename}`}
  alt={alt}
  width={1200}
  height={800}
  className="w-full object-cover"
/>
```

## Порядок подготовки (рекомендуемый)

1. **Email** — занимает 5 минут, разблокирует форму
2. **Скриншоты сайтов** (проекты 1–4) — за один подход, ~1 час
3. **Хопмалт** — организовать предметную съёмку
4. **Фото Галины** — студийная или хорошая домашняя
5. **og:image** — последним, 10 минут

После того как все файлы в `public/` — убрать плейсхолдеры.
